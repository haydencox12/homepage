Project 1

For my Database Systems and Applications class, I developed a database management application that provides a user-friendly interface for interacting with the database I was provided. The application supports all basic CRUD (Create, Read, Update, Delete) operations, allowing users to easily add new data to the database, read and retrieve existing data, update or modify data, and delete data as needed. This is achieved through a straightforward interface that guides the user through each operation step by step. It can connect to any database with some modifications, making it a versatile tool for managing a wide range of data types and structures. The project also includes error handling. Overall, this project demonstrates my solid understanding of database systems and applications.
